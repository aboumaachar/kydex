# NOTARY_USER_GUIDE

## Notary Workflow
1. Login to KYDEX portal.
2. Run screening for subject/entity.
3. Review match classification and risk.
4. Open/create case where escalation is required.
5. Attach evidence and export timeline when requested.

## Key Portal Pages
- /dashboard
- /screening/new
- /screening/document-extraction
- /cases
- /cases/{id}
- /cases/{id}/timeline
- /cases/{id}/evidence

## Best Practices
- Validate subject identifiers before submitting.
- Preserve chain-of-custody through evidence export flow.
- Escalate according to council governance rules.
