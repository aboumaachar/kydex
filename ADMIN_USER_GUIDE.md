# ADMIN_USER_GUIDE

## Admin Responsibilities
- Manage users and roles.
- Manage data sources and ingestion workflows.
- Monitor SLA breaches and system health.
- Review audit logs and export evidence.

## Core Admin Pages
- /admin/users
- /admin/roles
- /admin/data-sources
- /admin/ingestion-reports
- /admin/sla-breaches
- /admin/system-health
- /admin/audit-logs

## Operational Checklist
1. Verify preflight is OK.
2. Validate queue worker health.
3. Review failed ingestion events.
4. Validate recent backups.
5. Confirm no unresolved SLA breaches.
