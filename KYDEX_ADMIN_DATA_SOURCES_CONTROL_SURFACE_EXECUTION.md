# KYDEX — Admin Data Sources Control Surface (Design & Execution)

## 🎯 Objective

Design and implement a **fully controllable, auditable interface** for managing external and local watchlist sources (OFAC, UNSEC, etc.).

This page must allow administrators to:

- view all data sources
- trigger ingestion (sync)
- inspect versions
- review ingestion reports
- activate/archive versions
- verify data integrity (hash, origin, timestamp)

---

# 🧠 PRINCIPLE

```
If a regulator cannot see it, it does not exist.
```
