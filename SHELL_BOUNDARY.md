# KYDEX WORKSPACE BOUNDARY

Canonical URL:
https://kydex.me

Workspace:
kydex

Purpose:
This workspace is only for the KYDEX core search engine, compliance screening, APIs, authentication, database management, and original KYDEX service logic.

Allowed here:
- KYDEX landing page
- KYDEX dashboard
- KYDEX APIs
- OFAC / SDN / sanctions screening logic
- local database connectors
- user accounts and subscriptions
- search history and audit logs
- admin tools for KYDEX data integrity

Forbidden here:
- LCN / Notary Council institutional website content
- individual notary WordPress profile pages
- Council branding as the main product identity

Routing:
- Root domain: https://kydex.me
- API route: https://kydex.me/api
