# KYDEX Commercial Features

## Subscription and Plan Readiness

- Membership lifecycle states: TRIAL, ACTIVE, PAST_DUE, SUSPENDED, CANCELLED, EXPIRED
- Plan-aware manual and image usage limits
- Monthly usage counters per notary profile
- Billing period tracking fields

## API Key Governance

- Key create, rotate, suspend, activate, revoke
- Hash-only key storage
- Failed-auth tracking and abuse auto-suspend
- Per-key usage visibility

## Usage Controls

- Per key, site, IP, endpoint-type daily checks
- Per-plan monthly quota checks
- Deterministic rate-limit response contract

## Compliance and Audit

- Screening transaction logging
- Inquiry logging
- Source mode and fallback traceability
- Admin visibility dashboards for notary/subscription/usage posture

## Compliance Wording Guardrails

- Screening outputs are presented as decision-support signals only
- Final legal and compliance determinations require human review
- Product surfaces avoid definitive sanctions assertions for individuals
- Escalation language is used for review workflows instead of automated final outcomes
